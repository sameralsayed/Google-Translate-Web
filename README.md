# Google Translate Web

🌐 A sleek and fully functional web version of Google Translate with text, voice, photo, and conversation translation.

![Google Translate Web](https://via.placeholder.com/800x400/4285f4/ffffff?text=Google+Translate+Web)

## ✨ Features
- 📝 **Instant Text Translation** — 100+ languages with real backend
- 🔄 **Language Swap** — easy direction change
- 🎤 **Voice Input & Text-to-Speech**
- 📷 **Camera & Photo Translation** — simulation + upload
- 💬 **Conversation Mode** — bilingual chat simulation
- ⭐ **Phrasebook** — save and review favorite translations (localStorage)
- 📱 **Fully Responsive** — perfect on mobile, tablet, and desktop
- ⚡ **Fast & Clean UI** — modern Google-inspired design

## 🚀 How to Use
1. Open `index.html` in any modern browser
2. Type or speak text → click Translate
3. Use voice, camera simulation, or upload photo
4. Save useful phrases to Phrasebook
5. Try Conversation Mode for practice

## 🛠️ Built With
- HTML5
- CSS3
- Bootstrap 5
- jQuery
